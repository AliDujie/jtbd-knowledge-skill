## [1.0.0] - 2024-04

### Added | 新增

- **Core Python Package** (`jtbd/`)
  - `JTBDAnalyzer` - JTBD description and Four Forces analysis
  - `InterviewBuilder` - Dimension-based interview question generator
  - `ForcesProfile` - Structured Push/Pull/Anxiety/Habit analysis
  - `InnovationFinder` - Innovation signal identification and opportunity assessment
  - `config.py` - Global configuration and constants
  - `utils.py` - Knowledge base loading and text utilities
  - `templates.py` - Interview, report, and analysis templates

- **Knowledge Base** (11 markdown files)
  - `01-theory-foundation.md` - Theoretical foundations
  - `02-principles.md` - Core principles
  - `03-forces-of-progress.md` - Four Forces model deep dive
  - `04-system-of-progress.md` - Progress system
  - `05-research-methods.md` - Information gathering methods
  - `06-analysis-framework.md` - Information organization framework
  - `07-innovation-guide.md` - Innovation guide
  - `08-business-decisions.md` - Business decision making
  - `09-case-studies.md` - Case studies (Coffee vs Kale, SaaS activation, etc.)
  - `10-two-models.md` - Comparison of two JTBD models
  - `11-quick-reference.md` - Quick reference guide

- **Documentation**
  - Comprehensive README with bilingual (CN/EN) support
  - `SKILL.md` - AI Skill definition
  - `INSTALL.md` - Installation guide
  - `CONTRIBUTING.md` - Contribution guidelines
  - JTBD interview question library
  -实战 case studies with Four Forces analysis

- **Features**
  - Standardized JTBD statement format (Help me...so that...)
  - Four Forces analysis with intensity scoring
  - 8-dimension interview framework
  - Innovation signal identification (compensating behaviors, workarounds)
  - Opportunity assessment with feasibility/impact scoring
  - Knowledge base search functionality

### Technical Details | 技术细节

- Pure Python standard library (no external dependencies)
- Python 3.8+ compatible
- Cross-platform support (macOS, Linux, Windows)
- UTF-8 encoding for bilingual support
