# Contributing to JTBD Knowledge Skill

欢迎贡献！一起完善 JTBD 理论工具包。

## 🤝 如何贡献

### 1. 报告问题

创建 Issue 包含：
- 问题描述
- 复现步骤
- 预期行为
- 相关 JTBD 理论引用

### 2. 提交代码

```bash
git clone https://github.com/your-username/jtbd-knowledge-skill.git
git checkout -b feature/your-feature
# 修改代码...
git commit -m "feat: add new interview question template"
git push origin feature/your-feature
# 创建 Pull Request
```

### 3. 改进文档

- 补充 JTBD 理论说明
- 添加案例分析
- 改进中英文翻译
- 修正理论引用

## 📋 代码规范

- PEP 8 风格
- 完整 docstring
- 类型注解
- 清晰的命名

## 🧪 测试

- 新功能需测试
- 确保分析准确性
- 覆盖率 > 80%

## 💡 贡献方向

- 新增 JTBD 分析工具
- 改进四力分析框架
- 添加访谈问题模板
- 补充创新方法
- 优化案例研究

## 📚 理论依据

贡献需基于 JTBD 理论，推荐参考：
- 《When Coffee and Kale Compete》Alan Klement
- JTBD 官方资源

感谢贡献！🎯
